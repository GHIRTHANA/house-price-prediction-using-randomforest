This project is a House Price Prediction System built using the Random Forest algorithm in Machine Learning. The model predicts house prices based on property-related features such as area, number of bedrooms, bathrooms, floors, location, condition, garage availability, and year built. The project includes data preprocessing, feature encoding, model training, evaluation, and price prediction using Python and Scikit-learn.

Features:
1.House price prediction using the Random Forest algorithm
2.Data preprocessing and cleaning
3.Feature encoding and transformation
4.Model training and evaluation
5.Prediction based on property details

Technologies Used:Python,Pandas,Scikit-learn
